What You Need

1

Download MinerU2PPTX

Get the latest MinerU2PPTX.exe from the releases page. No installation required!

2

Prepare Your Files

You need two files:

  • Original File: PDF or image file you want to convert
  • MinerU JSON: Extracted data from MinerU service
3

Get MinerU JSON File

You have two options to extract the JSON data needed for conversion:

Option A: Web-based Extractor (Recommended)

Visit MinerU PDF Extractor website:

  1. Upload your PDF or image file
  2. Wait for processing to complete
  3. Download the generated JSON file
  4. Keep both files in an accessible location
Option B: Desktop Application

Download and install MinerU Desktop App (Windows):

  1. Install and run the application locally
  2. Process your PDF/image files to generate JSON data
  3. Export the JSON file for use with MinerU2PPTX
  4. Benefit from local processing and offline capability
âš ï¸ Important: The JSON file contains the structural data needed for accurate conversion. Without it, the conversion cannot proceed.
4

AI Setup (Optional but Recommended)

For the best results, configure AI text correction. This dramatically improves text quality, especially for:

  • Scanned documents with OCR errors
  • Non-English text
  • Complex formatting
  • Technical documents

How to Configure AI

1

Open AI Configuration

In MinerU2PPTX, click the "AI Config" button to open the configuration dialog.

2

Select Provider

Choose your preferred AI provider from the dropdown menu.

3

Enter API Key

Paste your API key in the designated field. Keys are stored locally and encrypted.

ğŸ” Security: API keys are stored in ai_config.json in your application directory. Never share this file publicly.
4

Test Configuration

Click "Test Connection" to verify your setup. The app will fetch available models automatically.

5

Select Model

Choose the specific model you want to use. Different models have different capabilities and costs.

API Key Storage

Your API keys are stored in ai_config.json in the application directory:

{ "providers": { "openai": { "api_key": "your-key-here", "model": "gpt-3.5-turbo" } }, "correction_enabled": true, "batch_size": 10 }

ğŸ“–How to Use

Single File Conversion

1

Launch Application

Double-click MinerU2PPTX.exe to start the application.

2

Select Input File

Click "Browse..." or drag and drop your PDF/image file into the first field.

3

Select JSON File

Click "Browse..." or drag and drop your MinerU JSON file into the second field.

4

Choose Output Location

The output path is auto-generated, but you can change it using "Save As...".

5

Configure Options

  • Remove Watermark: Automatically remove footers/headers
  • AI Text Correction: Enable if configured
  • Page Selection: Convert all pages or specify ranges
  • Generate Debug Images: For troubleshooting only
6

Start Conversion

Click "Start Conversion" and wait for completion.

Batch Processing

1

Switch to Batch Mode

Click the "Batch Mode" button in the top-right corner.

2

Add Tasks

Click "Add Task" to open the task configuration dialog.

3

Configure Each Task

For each task, specify the input file, JSON file, output location, and options.

4

Start Batch Conversion

Click "Start Batch Conversion" to process all tasks sequentially.
